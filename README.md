## Convention
### code
- setter 사용을 제한한다.
- 변수명은 CamelCase로 한다.
- 변수명이나 메서드명에 줄임말을 사용하지 않는다.
- 한 줄에 점을 2~3개 이내로 찍는다.
- 메서드와 메서드 사이에 공백 라인을 한 라인 둔다.
- 주석을 달아주자.
